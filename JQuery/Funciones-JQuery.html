Funcion Click

Clickea el boton para ejecutar el evento click

Funcion Hide, Show y Toggle

Clickea el boton para ejecutar el evento Hide o Show o Toggle

Funcion Slide Up, Slide slideDown y Slide Toggle

Desliza hacia arriba o hacia abajo, para ocultar o mostrar elementos

Funcion Fade In, Fade Out y Fade Toggle

Desvanece o Reaparece (disminuyendo o aumentando opcacidad)de un elemento

Funcion Add Class

Agrega una clase dinamicamente al elemento seleccionado.

Funcion After y Before

Clickea el boton para ejecutar el evento click

Funcion Prepend y Append

Agregar contenido antes o despues dentro de los elementos seleccionados (util en listas).

Funcion Html

Permite obtener codigo html o sobrescribirlo.

Obtiene o devuelve el contenido del primer elemento coincidente

Al asignar contenido, sobrescribe el contenido de todos los elementos coincidentes

Funcion Attr

Establece o devuelve atributos y valores de los elementos seleccionados.

Funcion Val

Devuelve o establece el atributo de valor de los elementos seleccionados(se usa principalmente en formularios html).


     

    

Funcion text

Establece o devuelve el contenido de texto de los elementos seleccionados.

Obtiene o devuelve el texto de todos los elementos coincidentes

Al asignar contenido, sobrescribe el contenido de todos los elementos coincidentes
